# A Universal Structural Framework for Intelligence

This is a speculative, unfinished idea. I am not actively working on it, and likely won't for several years. I'm putting it here so someone else can pick it up, critique it, formalize it, or prove it wrong.

**Read the full draft in [paper.md](paper.md).**

---

## Core Idea

Intelligence is not about how much information a system holds. It is about how much it can meaningfully traverse within its own structured knowledge, under constraint.

`I = C × Φ(S, St, W)`

Where:
- **I** = Intelligence (operational capacity)
- **S** = Stored Information
- **St** = Structure
- **W** = Weights
- **C** = Constraints
- **Φ** = The traversable inference space generated by S, St, and W

---

## Open Problems

### The Φ Problem

Find a rigorous, computable definition of Φ(S, St, W) — the traversable inference space — such that I = C × Φ yields results consistent with observed intelligence across different substrates.

### Subproblems

- **S Problem**: How to define and quantify a unit of information across different substrates?
- **St Problem**: How to classify and measure the structure of information? Which structures maximize traversability?
- **W Problem**: How to define "weight" substrate-independently? Does traversal itself modify weights?
- **C Problem**: How to formalize constraints? Which are hard, which are soft? Can a system rewrite its own constraints?
- **Time Problem**: How do S, St, W, C, and Φ evolve over time? Is there a governing equation for dΦ/dt?
